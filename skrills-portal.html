MCP tools are invoked by your AI assistant through the Skrills MCP server. Start the server with skrills serve (stdio) or skrills serve --http (HTTP), then register it in your CLI's MCP config. Click any tool to copy its name for use in MCP tool call requests.
CLI commands run in your terminal. Install Skrills via cargo install skrills or the install script, then paste any copied command into your shell. Click any command to copy it to your clipboard.
